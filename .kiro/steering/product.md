# Product Overview

## What is This?

A modern Windows desktop application for creating, editing, and previewing Mermaid diagrams and Markdown documents with real-time rendering. Built with WinUI 3, it provides a split-pane interface with a code editor and live preview panel.

## Primary Use Cases

- Creating UML diagrams (class, sequence, state, activity)
- Writing and previewing Markdown documentation with embedded Mermaid diagrams
- Exporting diagrams to PNG/SVG formats
- Exporting Markdown documents to Word (.docx) with rendered diagrams

## Key Features

- Real-time preview with <1 second update latency
- Syntax highlighting for Mermaid and Markdown
- Visual diagram builder for code-free diagram creation
- AI-powered diagram generation (OpenAI/Ollama integration)
- Customizable Markdown styling
- Recent files management
- Keyboard shortcuts and presentation mode

## Target Users

Developers and technical writers who need to create UML diagrams and technical documentation efficiently.
